## 📌 Project Overview
This project implements a machine learning–based system to detect fraudulent online payment transactions. It analyzes transaction patterns to accurately classify payments as legitimate or fraudulent, helping reduce financial risk and cyber fraud.

---

## 🧠 Key Highlights
- ✔ Data preprocessing & cleaning  
- ✔ Exploratory Data Analysis (EDA)  
- ✔ Handling class imbalance  
- ✔ ML model training & evaluation  
- ✔ Fraud prediction with performance metrics
